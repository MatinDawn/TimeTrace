# 留痕

留痕是一款基于微信小程序的个人记录与轻协作工具。

## 产品定位

以"留痕"为核心，覆盖：
- 已完成事项记录
- 未完成规划管理
- 花销记录与统计
- 习惯与趋势观察
- 多人空间协作

## 核心流程

1. 用户输入一句自然语言
2. 小程序创建云端草稿
3. AI 解析并回填结构化字段
4. 用户在确认页核对
5. 确认后正式提交

## 技术结构

**前端**：微信原生小程序，五个核心 Tab——首页 / TODO / 记账 / 统计 / 我的。

**后端**：
- 云函数 `liuhenBridge`：统一桥接所有读写
- 云开发 AI：自然语言解析与字段补全
- 云数据库集合：
  - `liuhen_users`
  - `liuhen_spaces`
  - `liuhen_space_members`
  - `liuhen_records`

## 协作模式

- **个人模式**：按微信身份隔离数据，默认只看自己的留痕
- **空间模式**：邀请码加入，同空间内共享数据，单空间上限 5 人
- 顶部导航支持随时切换

## 当前版本能力

已具备：
- 主流程：草稿 → AI 解析 → 用户确认 → 提交
- 个人 / 空间数据隔离
